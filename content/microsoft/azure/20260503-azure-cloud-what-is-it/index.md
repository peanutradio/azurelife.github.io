## Azure 클라우드란?

**Microsoft Azure**는 마이크로소프트가 운영하�� 글로벌 클라우드 컴퓨팅 플랫폼입니다. 전 세계 60개 이상의 리전에 데이터센터를 보유하고 있으며, 200개 이상의 서비스를 제공합니다.

## 왜 Azure인가?

- **엔터프라이즈 신뢰성**: Fortune 500 기업의 95%가 Azure를 사용
- **하이브리드 클라우드**: 온프레미스와 클라우드를 자연스럽게 연결
- **AI 통합**: OpenAI 모델을 Azure AI Foundry에서 바로 활용 가능
- **개발자 친화적**: Visual Studio, GitHub, VS Code와 긴밀한 통합

## 핵심 서비스 카테고리

### 컴퓨팅
- **Virtual Machines**: 윈도우/리눅스 가상 서버
- **App Service**: 웹앱 호스팅 (코드만 배포하면 끝)
- **Azure Kubernetes Service (AKS)**: 컨테이너 오케스트레이션

### 데이터 & 스토리지
- **Azure SQL Database**: 관리형 SQL 데이터베이스
- **Cosmos DB**: 글로벌 분산 NoSQL
- **Blob Storage**: ���용량 파일/미디어 저장

### AI & 머신러닝
- **Azure OpenAI Service**: GPT-4, GPT-4o 등 OpenAI 모델 API
- **Azure AI Foundry**: AI 앱 개발 통합 플랫폼
- **Cognitive Services**: 비전, 음성, 언어 AI API

### 네트워킹 & 보안
- **Virtual Network**: 가상 네트워크 구성
- **Azure Firewall**: 클라우드 방화벽
- **Microsoft Entra ID**: ID 및 접근 관리 (구 Azure AD)

## 시작하는 방법

1. [Azure 무료 계정](https://azure.microsoft.com/free/) 생성 (12개월 무료 서비스 + $200 크레딧)
2. Azure Portal에서 리소스 그룹 생성
3. 필요한 서비스 배포
